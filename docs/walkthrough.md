# ダイエット・モチベーションダッシュボード 完成しました！🚀

毎日の記録が楽しみになるような、あなた専用のプレミアムなダッシュボードが完成しました。

## 実装した機能・特徴

### 1. プレミアムなダーク・グラスモーフィズムデザイン
気分をアゲる「Neo-Tokyo / Cyberpunk」風のサイバーなカラーリング（ダークネオン系）に仕上げました。背景にはうっすらと光るグラデーションを配置し、各パネルはすりガラスのような透け感（グラスモーフィズム）を持たせています。

### 2. 進捗が一目でわかるKPIパネル
「今の自分は目標に届いているのか？」が瞬時にわかるように、以下の4つの主要KPIを配置しました。
* **現在の体重**: スタートからの減量幅も連動表示されます。
* **目標まで残り**: 目標の73.1kgまであと何kgかを表示します。
* **本日の目標ライン**: 「スタートからゴールまで一切の停滞なく痩せ続けた場合の、今日あるべき体重」を毎日自動計算します。現在体重がここを下回っていると、緑色の文字で「**オンスケジュール！🔥**」と褒めてくれます。
* **残り日数**: 84日間のうち、本日は何日目かをカウントダウンします。

### 3. Recharts を用いた進捗グラフ
* 紫色の直線（理想ライン）と、水色の折れ線（実績ライン）が重なるように表示されます。
* 実績ラインが紫色のラインより**下**にいれば、順調な証拠です！

### 4. モチベーション向上機能（お祝い機能）
* 本日の体重を入力し、「前回の記録より減っていた」場合、画面全体に **NICE DROP! 🎉** というお祝いアニメーション（紙吹雪風）が表示されます。日々のモチベーション維持に貢献します。

### 5. セキュア＆完全ローカル動作
バックエンドサーバーやデータベースを構築せず、入力された体重データは**ブラウザのローカルストレージにのみ保存**されます。情報漏洩の心配がなく、GitHub Pagesにも簡単に展開（デプロイ）可能です。

## 起動方法・使い方

ターミナルを開き、以下のコマンドでお手元で起動できます。

```bash
# パッケージのインストール（まだ完了していない場合）
npm install

# ローカルサーバーの起動
npm run dev
```

起動したら、コンソールに表示されるURL（例: `http://localhost:5173/`）にブラウザでアクセスしてください。初期状態では、4月16日時点までのこれまでの実績データがセットされていますので、そのまま明日の記録へ進めます。

「更新してぶち上げる！」ボタンで、毎日の記録を叩き出してください！🔥
