# 変更履歴 (Changelog)

すべてのバージョンの変更履歴（システムの進化）をここに記録します。

## [v1.0.0] - 2026-04-16
### 追加 (Added)
* **初期ダッシュボードの構築**: React / Vite / LocalStorage を用いたバックエンド不要のSPA構成
* **KPIパネル**: 現在の体重、目標までの残り、本日の目標ライン、残り日数を即座に表示
* **進捗グラフ**: Rechartsを用いた平均目標ライン(紫)と実績ライン(水色)の可視化
* **モチベーション機能**: 前日の記録より体重が減少していた場合のお祝いアニメーション（紙吹雪）
* **レコード削除機能**: 履歴リストのゴミ箱ボタンからの特定日時の記録の安全な削除
* **CI/CD環境**: GitHub Actions (`deploy.yml`) を用いた GitHub Pages への完全自動デプロイ
* **各種ドキュメント**: システム設計書、初心者向け解説ガイド、バックログ管理表の策定
* **プロジェクト管理機能**: ADR（検討ログ）ベースの意思決定プロセスやブランチ管理ルールの導入
